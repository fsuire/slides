Profiling

Pour lancer le profiler:

node --prof src/index.js

Cela va créer un fichier de type isolate-xxxx-v8.log, relativement illisible pour un humain. Heureusement une commande permet de le rendre plus digeste :

node --prof-process isolate-xxx-v8.log > processed.txt

Ces deux fichiers peuvent être ajoutés au .gitignore

Très peu utilisé, njsTrace est pourtant une alternative intéressante.
