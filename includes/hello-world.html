Créerun nouveau projet

# Créer le dossier qui accueillera le projet
mkdir formation
cd formation

# Wizard d'initialisation de projet Node.js
# Peut importe ce que l'on répond aux questions posées,
# tout pourra être changé facilement par la suite
npm init

Aprés avoir exécuté ces commandes, deux nouveaux fichiers ont été créés dans le répertoire de votre projet :

  • Le fichier package.json, qui contient la configuration basique de votre projet ainsi que la liste de ses dépendances
  • Le fichier package-lock.json, qui est le fichier de verrouillage des versions des dépendances généré par NPM

Créerun premier fichier JavaScript

Toutes nos sources se trouveront dans le dossier src. Cela permettra de les isoler facilement des fichiers de configurations du projet.

Par convention, un fichier index.js sert de point d'entrée au répertoire dans lequel il se trouve. Nous allons donc créér le fichier src/index.js

"use strict";

console.log('Hello world!');

Et pour faire tourner ce code :

node src/index.js
