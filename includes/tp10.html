Travail pratique #10

À partir de master, tenter de développer, à plusieurs, une interface web pour le jeu de morpion. À défaut de le faire réellement, un bon échange sur les multiples sujets qui y sont liés devrait être profitable.

Quelques pistes pour commencer :

  • Penser à une architecture permettant de rendre les routes testables. Le guide du routage de express peut être utile.
  • Pour l'utilisation des données de formulaire : body-parser
  • Un serveur statique ?
  • Un moteur de template ?
